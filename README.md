# 🛡️ Sistema de Previsão Aegis

Este é um sistema de inteligência preditiva desenvolvido para otimização de recursos e planeamento estratégico. O projeto utiliza modelos avançados de séries temporais para prever a procura futura com alta precisão.

## 📖 Documentação
Para entender como o sistema funciona, os modelos utilizados e como interpretar os resultados, consulte o ficheiro:
👉 **[DOCUMENTACAO.md](./DOCUMENTACAO.md)**

## 🛠️ Tecnologias
- **React 19**
- **Vite 7**
- **Recharts** (Gráficos)
- **Lucide React** (Ícones)

## 🚀 Como Executar
1. Instale as dependências: `npm install`
2. Inicie o servidor: `npm run dev`
